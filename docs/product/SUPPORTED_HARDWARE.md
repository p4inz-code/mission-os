# Supported Hardware

## Architecture

- x86_64 (Primary)

Future consideration

- ARM64

---

## Boot

- UEFI (Primary)

Legacy BIOS support may be considered separately.

---

## Storage

- USB 3.0+
- SATA SSD
- NVMe SSD

---

## Memory

Minimum

- 8 GB RAM

Recommended

- 16 GB RAM

---

## CPU

Minimum

- Modern 64-bit processor

Recommended

- 4+ physical cores

---

## GPU

Supported

- Intel
- AMD
- NVIDIA

---

## Network

- Ethernet
- Wi-Fi
- Bluetooth

---

## Unsupported

- 32-bit processors
- End-of-life hardware
- Unsupported proprietary drivers
