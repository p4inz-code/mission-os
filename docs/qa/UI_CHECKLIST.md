# UI Checklist

## Visual Consistency

- [ ] Color palette follows the Design Bible.
- [ ] Typography follows the Design Bible.
- [ ] Iconography is consistent.
- [ ] Spacing uses approved spacing scale.
- [ ] Corner radius is consistent.
- [ ] Shadows follow design guidelines.
- [ ] Borders are consistent.
- [ ] Motion follows animation guidelines.

## Layout

- [ ] No overlapping elements.
- [ ] Responsive across supported resolutions.
- [ ] Safe margins maintained.
- [ ] Proper alignment throughout.
- [ ] Scroll behavior verified.
- [ ] Window resizing behaves correctly.

## Interaction

- [ ] Buttons provide hover feedback.
- [ ] Buttons provide pressed feedback.
- [ ] Disabled states are visible.
- [ ] Loading states exist.
- [ ] Error states exist.
- [ ] Empty states exist.
- [ ] Success feedback is clear.

## Accessibility

- [ ] Keyboard navigation complete.
- [ ] Focus indicators visible.
- [ ] Screen reader labels verified.
- [ ] Color contrast meets WCAG AA.
- [ ] Text scaling supported.
- [ ] High contrast mode verified.

## Desktop Integration

- [ ] Window decorations correct.
- [ ] Window controls function correctly.
- [ ] Context menus verified.
- [ ] Notifications display correctly.
- [ ] Drag-and-drop behaves correctly.

## Performance

- [ ] No visible UI stutter.
- [ ] No animation tearing.
- [ ] Startup animation smooth.
- [ ] Memory usage acceptable.

## Validation

- [ ] Design review approved.
- [ ] UX review approved.
- [ ] Accessibility review approved.
